Entendendo Comunicação Client x Server

Nessa seção estudamos um pouco sobre a história da web, a história da internet e seus principais nomes. Vimos também o que são clients e o que são servers, conversamos sobre navegadores, aplicações web e dispositivos móveis e vários outros temas.
Agora iremos, de forma sucinta, entrar em detalhes sobre tudo o que foi estudado.

História da Web

Nessa seção conversamos sobre os computadores de 1ª geração, o Harvard Mark I (1944), Colossus (1946) e ENIAC(1946), também sobre os de 2ª geração (1959 a 1965) que foi onde os computadores diminuiram bastante de tamanho pois passaram a utilizar transitores. Depois entramos com os computadores de 3ª geração (1965 a 1970), onde eles já tinha ligeira semelhança com os que temos hoje, foi ai que os computadores passaram a se utilizar de chips, um circuito integrado composto por transitores, capacitores e resistores. Finalmente chegamos na geração em que estamos hoje, a 4ª geração (a partir de 1971), foi aqui onde a tecnologia como conhecemos hoje se desenvolveu.

Vimos também que é dito que já se diz que entramos na 5ª geração, esta que seria composta por Inteligências Artificiais(AI), criptomoedas, metaverso e supercomputadores.

Após falarmos sobre os computadores, entramos no tema de como surgiu a Internet.
A internet surgiu na década de 1950, durande a Guerra Fria, quando se viu a necessidade de compartilhar dados entre computadores, foi assim que surgiu a DARPA onde era possível compartilhar dados entre computadores que entivessem no mesmo prédio através de cabos.

Logo após falamos sobre o Joseph Carl Robnett Licklider que teorizou sobre uma rede onde todos poderiam se comunicar através dela. Um dos grandes nomes quando se fala do surgimento da Internet.

Após isso surgiu a ARPANET que era um sistema de comunicação em pacotes, enviados um a um e que permitia múltiplos destinos. A primeira conexão estabelecida foi feita entre a Universidade da Califórnia e o Instituto de Pesquisas de Stanford na data de 29/10/1969, onde foi enviada a palavra "LOGIN", porém foi recebido apenas o "LO", sendo assim, se deu tanto a primeira conexão quanto a primeira queda de conexão.

Os grandes nomes da tecnologia que estudamos até o momento foram:

Grace Murray Hoppe: Uma das primeiras programadoras do Mark I em 1944, foi criadora da linguagem de programação de alto-nível Flow-Matic(base do COBOL), bem como criadora do termo 'bug' devido aos insetos que os primeiros computadores costumavam atrair devido ao seu calor.

Joseph Carl Robnett Licklider: Recrutado pela DARPA depois de teorizar sobre uma rede 'galática' de computadores. Plantou a semente da comunicação entre dois pontos distintos através do computador.

Robert E. Kahn: Primeira apresentação pública da ARPANet, apresentou o primeiro e-mail e foi o criator do protocolo TCP-IP.

Tim Berners-Lee: Conhecido como Pai da Internet, foi ele que criou o WWW,é diretor do W3C, recebeu um título de cavaleiro dados pela Rainha Elizabeth II, e foi ele quem popularizou o HTTP e o HTML.

Marc Andreessen: (Re)inventou o navegador criando o Netscape que foi feito a partir do Mosaic, tornou o navegador 'amigável', utilizando recursos gráficos. Ele chegou a deter 90% da internet da época, sendo detido pelo Internet Explorer anos depois.

Bill Gates: É o fundador da Microsoft, com que criou o Internet Explorer. Foi ele quem tornou acessível para todos o computador e, com isso o acesso á internet.

Steve Jobs:É o fundador da Apple que é concorrente direta da Microsoft. Foi ele quem popularizou o smartphone que conhecemos hoje através do lançamento do iPhone em 2007.

O que são Clients?

Um cliente é um computador ou outro dispositivo que interage com um servidor para aceder ou trocar dados. É tipicamente utilizado para se referir a um dispositivo que solicita dados de um servidor, tal como quando um navegador web solicita uma página web de um servidor web.

O que são servers?

Como definição rudimentar, podemos considerar um servidor como qualquer solução tecnológica com capacidade para executar programas de forma centralizada, além de armazenar e compartilhar arquivos, administrar filas de impressão e prestar algum tipo de serviço para vários computadores, através de uma rede local ou remota. Esses equipamentos geralmente são os responsáveis por atender as requisições dos computadores clientes da rede local. Um servidor local têm como função centralizar as aplicações corporativas de TI dentro das organizações, executar softwares administrativos e banco de dados.

Um servidor pode ser:

Servidor de Aplicação: São utilizados para executar aplicações corporativas e atender diversas estações de trabalho de forma simultânea, os servidores de aplicação geralmente necessitam de seu poder de processamento para executar programas que não podem ser executados apenas numa estação de trabalho.

Servidor de Arquivos: Servidores de arquivos são sistemas para armazenar e compartilhar uma grande quantidade de informações entre equipamentos e usuários, mantendo o gerenciamento centralizado.

Servidor de Banco de Dados: Alguns equipamentos são configurados para potencializar e dar eficiência, de forma dedicada, ao processamento e a transferência de dados entre os demais sistemas computacionais da infraestrutura de TI, como alguns servidores de aplicação e storages.Esses computadores também são conhecidos como servidores de banco de dados e normalmente proporcionam um ambiente com desempenho apropriado para instalar e processar bases de dados que recebem um grande número de requisições.

Servidor de Mídia: Também conhecido como media servers, esses equipamentos podem transmitir conteúdo de áudio ou vídeo via internet, através de um processo conhecido como streaming. Netflix, Amazon Prime, Youtube e outras empresas de conteúdo são exemplos de hard users desse tipo de tecnologia.

Servidor de e-mail: Servidores de email são equipamentos configurados para armazenar e transferir e-mails através de redes locais ou via internet.

Linguagens de Progamação

Uma linguagem de programação é por onde o hardware (máquina) e o programador se comunicam. É uma linguagem formal que funciona por meio de uma série de instruções, símbolos, palavras-chave, regras semânticas e sintáticas.A linguagem de programação permite que um programador crie programas a partir de um conjunto de ordens, ações consecutivas, dados e algoritmos.

As linguagens de programação podem ser de Alto nível ou de Baixo nível. As linguagens de Baixo nível são aquelas que estão mais próximas da linguagem de máquina, os números binários. Já as de Alto nível são aquelas que estão mais próximas da linguagem humana, normalmente tendo base no inglês.

HTML é linguagem de programção?

Não, o HTML não é uma linguagem de programação.

O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.
